# 🎓 Registration Form

This project is a responsive **Registration Form** that validates user input in real time using JavaScript. Users must enter a valid **Name**, **Email**, and **Password** before the **Submit** button becomes active. Once all fields are valid, the form can be submitted successfully, displaying a confirmation message and resetting the form.

---

## 🚀 Features

* ✅ Real-time form validation
* ✅ Name field validation
* ✅ Email format validation
* ✅ Password validation (minimum 6 characters)
* ✅ Dynamic error messages
* ✅ Submit button enabled only when all fields are valid
* ✅ Success message after successful registration
* ✅ Automatic form reset
* ✅ Responsive design for desktop and mobile devices
* ✅ Clean and beginner-friendly user interface

## 🎯 Validation Rules

### 👤 Name

* Cannot be empty.

### 📧 Email

* Cannot be empty.
* Must contain `@` and `.`.

### 🔒 Password

* Must contain at least **6 characters**.
